# Краткое введение
В данной дипломной работе исследуется рост биопленок на поверхности нержавеющей стали.

Структура диплома:
1. Введение
2. Постановка задачи
3. Актуальность
4. Обзор литературы
5. Методология
6. Результаты

Мы занимаемся только обзором литературы сейчас.
Предлагаю использовать следующую струтуру, по которой доставать из взятых статей
необходимую информацию:
1. Ссылка на статью (по ГОСТУ + отдельно пихнуть DOI)
2. Постановка эксперимента:
	1. Цель
	2. Материалы (вид нержавейки, какое молоко)
	3. Условия проведения эксперимента (температура, влажность, время проведения)
4. Другие характерные особенности и заметки (например, 
чем может быть полезна данная статья с точки зрения особенностей эксперимента и 
препаратов).

Соответственно, можно построить обзор литературы по следующим шаблонам:
1. Группировать работы по коллективам авторов
2. Группировать по подходам

Так как часто одна группировка применяет один подход, со временем развивая его,
можно комбинировать данные шаблоны. Но сначала статьи. Читать.
